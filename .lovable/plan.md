# Milestone: AI Wellness — Build Plan

Single-page TanStack Start app (uses existing `src/routes/index.tsx`) — a bilingual journaling + reflection tool. No backend, no auth, all local state. Calm-tech aesthetic with EN ⇄ हिंदी toggle.

## Generic, not Riya-specific
The app boots in a generic Demo State — no hardcoded persona. The user sets their own context in a lightweight "About you" panel that drives the reflection content.

- On first load, the header greets a friendly default ("Welcome back") with no fixed name.
- A small **Your context** card (collapsible, above the journal) lets the user fill optional fields locally:
  - Name (free text)
  - Role / life stage (free text, e.g. "student", "designer", "parent")
  - Current focus (free text, e.g. "exam prep", "job switch", "burnout")
- These values live in `useState` (and `localStorage` for convenience). They are interpolated into the reflection cards so the output adapts to whoever is using it — no persona is assumed.
- If the user leaves the context empty, cards fall back to neutral, universal wording.

## Reflection content (dynamic, deterministic — no hallucinations)
Analysis is a simulated 1.2s delay, then renders three cards. Content is generated by simple **rules over the journal text + context**, not an LLM:

- **Card 1 — Triggers (tag chips):** keyword scan over the journal text maps to a fixed bilingual tag library (e.g. words like "exam/deadline/score" → "Performance Pressure"; "tired/burnout/overwhelmed" → "Burnout Signals"; "lonely/alone" → "Social Disconnect"; "future/career" → "Future Uncertainty"). If nothing matches, show a neutral "General Reflection" tag.
- **Card 2 — Contextual Reassurance:** a templated paragraph that addresses the user by name if provided and references their role/focus if provided, e.g. *"Your current focus on {focus} is a chapter, not the whole story. The effort you're putting into {role} matters, even when progress feels invisible."* Falls back to a fully generic version when fields are empty.
- **Card 3 — Actionable Coping:** picked from a small fixed list keyed off detected stress level (count of matched trigger keywords): low → grounding suggestion; medium → 5-minute movement break; high → step away from screens + breathing exercise. Each option is pre-translated.

Every string lives in `src/components/wellness/i18n.ts` keyed `en`/`hi`, so toggling language re-renders instantly — including the dashboard.

## Components (`src/components/wellness/`)
- `LanguageToggle.tsx` — EN | हिं pill, `aria-pressed`, keyboard accessible.
- `Header.tsx` — app title (serif), tagline, language toggle.
- `ContextPanel.tsx` — optional name/role/focus inputs, collapsible.
- `JournalInput.tsx` — labeled `<textarea>` + Analyze button.
- `ReflectionDashboard.tsx` — renders the three cards after analysis.
- `ReflectionCard.tsx` — shared card shell (variant: triggers/reassurance/coping).
- `Footer.tsx` — permanent crisis helpline disclaimer.
- `i18n.ts` — all strings + trigger keyword tables.
- `analyze.ts` — pure function `analyze(text, context, lang) → { triggers, reassurance, coping }`.

## Behavior
1. Boot: `language="en"`, empty context, empty journal.
2. User optionally fills context, types in textarea, clicks Analyze.
3. Loading state (~1.2s, `aria-live="polite"`) → dashboard renders, smooth scroll into view.
4. Language toggle instantly re-renders all visible strings.
5. Re-analyze anytime; results update.

## Aesthetic (Pinterest Calm Tech)
- Soft sage background `#E8F3E8`, warm off-white cards `#FAFAFA`, dark slate text — added as oklch tokens in `src/styles.css` via `@theme inline`.
- Serif display (Fraunces) + sans body (Inter), loaded via `<link>` in `src/routes/__root.tsx`, registered as `--font-serif` / `--font-sans`.
- `rounded-2xl`, soft multi-layer floating shadow utility, subtle `backdrop-blur` for glassmorphism.
- No harsh colors; muted sage accents only.

## Accessibility
- Semantic `<header>`, `<main>`, `<section aria-labelledby>`, `<footer>`; single `<h1>`.
- Toggle buttons use `aria-pressed`; textarea has a real `<label htmlFor>`.
- Loading region uses `aria-live="polite"`.
- Visible focus rings via `--ring`; tap targets ≥ 44px; WCAG AA contrast on both color pairs.

## Files touched
- `src/routes/index.tsx` — compose page, hold state, set `head()` (title "Milestone: AI Wellness", meta description).
- `src/routes/__root.tsx` — add Google Fonts `<link>` for Fraunces + Inter.
- `src/styles.css` — sage palette tokens, font tokens, soft-shadow utility.
- New files under `src/components/wellness/`.

## Out of scope
- No routing beyond `/`. No server persistence. No real AI call. No login. No hardcoded persona.
